# Amazon Clone

A frontend clone of the Amazon homepage built using **HTML** and **CSS**. This project focuses on recreating the user interface of Amazon while improving my frontend development skills.

## Features

- Responsive navigation bar
- Search bar
- Hero/banner section
- Product category cards
- Footer similar to Amazon
- Clean and organized layout
- Responsive design 

## Technologies Used

- HTML5
- CSS3

## Purpose

This project was created to practice:
- Semantic HTML
- CSS Flexbox and Grid
- Responsive web design
- Layout structuring
- Frontend development fundamentals

## Disclaimer

This project is created for educational purposes only. It is a frontend clone and is not affiliated with or endorsed by Amazon.
